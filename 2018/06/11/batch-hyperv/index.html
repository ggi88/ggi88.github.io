2018-06-11

Hyper-V 활성/비활성

  • 윈도우에서 vmware와 docker를 같이 사용할 때 충돌이 발생 할 수 있기에 사용에 따라 활성, 비활성화 해야한다.
  • Hyper-V / Vt-x 동시 사용불가
  • 활성화: bcdedit /set hypervisorlaunchtype auto
  • 비활성화 : bcdedit /set hypervisorlaunchtype off
  • 설정 후엔 재부팅 필요
